# 🔮 Oracle of the Pitch

> *The cosmos has already decided. We merely translate.*

A single-page web app that generates gloriously absurd FIFA World Cup 2026 match predictions using ancient celestial wisdom, suspicious pigeons, and seventeen incompatible prophecy systems.

Built for entertainment during the 2026 FIFA World Cup (USA, Canada & Mexico).

## What it does

- Select any two of the 48 World Cup 2026 nations
- Consult the Oracle
- Receive an unhinged, cosmically-justified match prediction
- Download or share the prophecy card with friends

## Features

- All 48 qualified nations with real player squads
- Star-weighted player selection (famous names appear more often)
- 25 unique prophecy templates across different comedic tones
- Shareable prediction card image via html2canvas
- Mobile-first design
- Zero backend — runs entirely in the browser

## Tech

Pure HTML, CSS, and JavaScript. No framework, no build step, no dependencies except the Google Fonts and html2canvas CDN links in the `<head>`.

To run locally just open `index.html` in a browser.

## Deployment

Deployed as a static site on Vercel: [oracleofthepitch.vercel.app](https://oracleofthepitch.vercel.app)
